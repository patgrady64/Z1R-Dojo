Z1 Dojo Principles
1. Practice First — Every feature must improve deliberate practice.
2. Fast Reset — Never make players replay content unnecessarily.
3. Instant Access — Any drill should start within seconds.
4. Measure Improvement — If we can track progress, we should.
5. Respect the Original Game — Keep Zelda's mechanics authentic.
6. Code Quality Matters — Clean architecture over quick hacks.
7. Document Everything — Every discovery should be written down.
8. Community Driven — Build features that solve real player problems.
